<h2>What is the box model?</h2>
<p>Don‚Äôt worry, it‚Äôs not food (just kidding üòÖ). To understand the box model, you should know that every visible HTML element on the page has four parts. When combined, they form the box model.</p>
<h4>So what are these parts?</h4>
<p>They are:</p>
<ul>
<li><b>content:</b> the space taken by the content (text, children, etc.)</li>
<li><b>padding:</b> the space between the content and the border</li>
<li><b>border:</b> the line that surrounds the padding and content, which can be styled (color, width, type)</li>
<li><b>margin:</b> the space outside the border, separating the element from others</li>
</ul>

<h2>Box-sizing property</h2>
<p>To avoid confusion or miscalculation, we must be precise. The box-sizing property has two main values:</p>
<ul>
<li><b>content-box:</b> the default. Width and height apply only to the content. Padding and border are added outside.</li>
<li><b>border-box:</b> width and height include content, padding, and border.</li>
</ul>
<div class="exp_2">
    <div class="exp_2_1">
        <code>&lt;style&gt;<br>
        parent{ padding:10px; box-sizing: content-box; height:auto; width:auto; }
        <div class="cmt">/* calculates only the content‚Äôs width and height */</div>

        img {  width: 100%;height: auto;border:2px solid #0682e8ff; }

        <div class="cmt">/* the total width and height exclude padding and border, so the image may overflow the parent */</div><br><br>
         parent{ padding:10px; box-sizing: border-box; height:auto; width:auto; }
        <div class="cmt">/* calculates the total size including padding and border */</div>

        img {  width: 100%;height: auto;border:2px solid #0682e8ff; }

        <div class="cmt">/* in this case, the border is included in the element's total width and height, so the image fits perfectly inside the parent */</div>
    </div>
    <div class="exp_2_2">
        <div class="real_exp" style="box-sizing: content-box;height:100px;margin: 10px;">

<h2>Box model best practices</h2>
<p>For effective box model management, follow these best practices:</p>
<ul>
<li class="prct">Use the box-sizing property</li>
<li class="prct">Reset margin and padding for elements to avoid unexpected spacing</li>
<li class="prct">Use shorthand properties for cleaner code (e.g., margin: 10px 20px;)</li>
<li class="prct">Be aware of default browser styles and use a CSS reset if needed</li>
<li class="prct">Test layouts across different browsers to ensure consistency</li>
</ul>
<h2>üéÅ Bonus Tip:</h2>
