<h2>Display property</h2>
<p>So, what is the display property? In simple words, it controls the space a tag takes up. Not like the box model (which is about spacing inside and around elements), but how the element itself behaves on the page. Tags are divided into two main types:</p>
<ul>
<li><b>1- Block elements:</b> take the whole line space (p, div...)</li>
<li><b>2- Inline elements:</b> take only the space of their content on the line (b, span...)</li>
</ul>

<h2>Position property</h2>
<p>The position property defines how an element is placed on the page. By default, tags are positioned in the normal document flow (static). But we can change this behavior. Here are the most important values:</p>

<div class="tbl_area">
 <table class="tbl_2">
 <tr><td class="tbl_2_h">Values</td><td class="tbl_2_2_h">Explanation</td></tr>
<tr><td class="tbl_2_1">static</td><td class="tbl_2_2">Default position (elements follow normal document flow)</td></tr>
<tr><td class="tbl_2_1">relative</td><td class="tbl_2_2">Positioned relative to its normal position</td></tr>
<tr><td class="tbl_2_1">absolute</td><td class="tbl_2_2">Positioned relative to the nearest positioned ancestor</td></tr>
<tr><td class="tbl_2_1_d">fixed</td><td class="tbl_2_2_d">Positioned relative to the viewport, does not move when scrolling</td></tr>
</table>
</div>

<h2>Flexbox Basics</h2>
<p>Flexbox is a modern and powerful layout system in CSS. It makes it much easier to arrange elements in rows or columns, and to align or distribute space between them. It‚Äôs flexible (hence the name) and widely used today.</p>

<div class="exp">
        <code>
        .boxflex{<br>
            display: flex;<br>
           <div class="cmt"> /* to enable flex context and be a flex box */</div><br>
        }
        </code>
</div>

<h2>üéÅ Bonus Tip:</h2>
<ul>
  <li>We use sticky position for section headers and table column labels, and fixed for sticky headers, floating buttons, or alerts. In short, both are useful for simple animations or page decorations.</li>
  <li>Display inline-block is used when you want to control the size and position of an inline tag.</li>
  <li>Float works mostly with text and images. It‚Äôs often used in news websites where text wraps around images.</li>
  <li>Clear is used when you want to stop a specific element from moving next to a floating element.</li>
  <li>Clear should be applied to a block element to move it to the next line.</li>
  <li>Flexbox is a large topic that we will cover in the next course.</li>
</ul>
