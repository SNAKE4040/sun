   
    <h1>Introduction:</h1>
    <p>CSS is a powerful tool that allows you to style your HTML documents. It helps you create visually appealing web pages by controlling layout, colors, fonts, and more. Before learning CSS, it is important to have a basic understanding of HTML.</p>
    <p>CSS stands for <b>Cascading Style Sheets</b>. It is a stylesheet language used to describe the presentation and layout of HTML documents. CSS enables developers to control the appearance and design of web pages across different devices and screen sizes. It was first proposed by H√•kon Wium Lie in 1994 and has since evolved through multiple levels (CSS1, CSS2, CSS3) to include a wide range of styling features and responsive design capabilities.</p>

    <h2>How to Use CSS</h2>

    <h3>1. External CSS</h3>
    <p>You can write CSS in a separate file (e.g., <b>style.css</b>) and link it to your HTML using the <b>&lt;link&gt;</b> tag inside the &lt;head&gt; section.</p>
    <div class="exp">
        <code>&lt;link href="style.css" rel="stylesheet"&gt;</code>
    </div>
    <p><i>The <b>href</b> attribute specifies the path to the CSS file.</i></p>

    <h3>2. Internal CSS</h3>
    <p>You can write CSS code directly inside your HTML file within a &lt;style&gt; tag, usually placed in the &lt;head&gt; section.</p>
    <div class="exp">
        <code>&lt;style&gt;<br>
        p { color: blue; }<br><br>
        &lt;/style&gt;</code>
    </div>

    <h3>3. Inline CSS</h3>
    <p>CSS can also be applied directly to an individual HTML element using the <b>style</b> attribute.</p>
    <div class="exp">
        <code>&lt;p style="color: green;"&gt;This is a styled paragraph&lt;/p&gt;</code>
    </div>

    <h3>üéÅ Bonus Tip:</h3>
    <p><b>When to use each method:</b></p>
    <ul>
        <li><b>External CSS</b>: Best for large websites with multiple pages sharing the same styles.</li>
        <li><b>Internal CSS</b>: Useful for single-page documents or when custom styles are needed for one HTML file.</li>
        <li><b>Inline CSS</b>: Used for quick styling of individual elements. Not recommended for large-scale styling.</li>
    </ul>

    <div style="color: green; font-size: 1.5em;">Text in paragraph</div>

    <p>CSS follows a priority order called the <b>Cascade</b>: Inline styles override internal styles, and internal styles override external styles (unless <code>!important</code> is used).</p>
